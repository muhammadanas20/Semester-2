## KNOWN TEST SCENARIOS & VARIATIONS

### Scenario 1: Multi-Part Boolean Problem
**Year Variant**: 2025, 2020, 2023
**Structure**:
- Part A: Simplify expressions using K-Map (3 marks)
- Part B: Implement using specific gates (3 marks)
- Part C: Design real-world circuit (4 marks)
**Total**: 10 marks
**Tip**: Start with part C logic, then reduce to parts A & B

### Scenario 2: Personalized Counter Problem
**Year Variant**: 2020 (Q4b), 2023
**Structure**: 
- Roll number determines sequence
- No two students have identical problem
- Tests understanding, not memorization
**Marks**: 20 marks (within 40-50 mark counter section)
**Tip**: Understand the process → generic enough to apply to any sequence

### Scenario 3: Hierarchical Decoder Design
**Year Variant**: 2020 (Q2d)
**Structure**:
- Build larger decoder from smaller modules
- Minimize component usage
- No internal circuit diagrams required
**Marks**: 5 marks
**Tip**: Calculate total outputs → work backwards to module count

### Scenario 4: Comprehensive Counter Design (Full Process)
**Year Variant**: 2025 (Q5a), 2020 (Q4b-c)
**Structure**:
- State table given or implied
- K-Map design of each flip-flop
- Circuit diagram (gates specified)
- Timing diagram (≥8 cycles minimum)
**Marks**: 6-20 marks
**Tip**: Most time-intensive; allocate 35+ minutes

### Scenario 5: Shift Register Modes
**Year Variant**: 2020 (Q5a), 2025 (Q6c)
**Structure**:
- Mode table with select inputs (S0, S1)
- Four modes: Load, Shift, Hold, Clear
- Draw logic for each mode
**Marks**: 3-4 marks
**Tip**: Multiplexer-based approach most efficient

## KEY FORMULA & REFERENCE SHEET

### Conversion Formulas
```
Decimal to Binary: Divide by 2, collect remainders (LSB first)
Binary to Decimal: ∑(bit_value × 2^position)
Decimal to Gray: Binary → shift left, XOR with original
Gray to Binary: Work from MSB, XOR each bit with previous result
2's Complement: Flip all bits + add 1 (for negative representation)
```

### Counter State Formula
```
For n-bit binary counter: 2^n maximum states
Modulo-N design: Need (2^n - N) states to reset
Johnson counter states: 2n (vs Ring: n)
Ring counter period: n clock cycles minimum
```

### Gate Equivalencies (Universal Logic)
```
NAND-NAND Implementation:
- OR gate: NAND(NOT(A), NOT(B)) = NAND(A', B')
- NOT gate: NAND(A, A)
- AND gate: NOT(NAND(A, B))

NOR-NOR Implementation:
- AND gate: NOR(NOT(A), NOT(B)) = NOR(A', B')
- NOT gate: NOR(A, A)
- OR gate: NOT(NOR(A, B))
```
